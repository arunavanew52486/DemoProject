# DemoProject

## Java Backend
Spring Boot
Testing: Mockito

## Angular Frontend
### Technologies used:
	-***Frontend Framework:*** Angular
 	-***Styling:*** Tailwind CSS, Flowbite
	-***Testing:*** JEST

### How to run?
	-Go to the ***/AngularFrontend*** folder.
 	-Open the terminal (make sure nodejs & angular is installed)
	-run ***npm i***
 	-run ***ng serve***
	
### How to run test cases?
	-Go to the ***/AngularFrontend*** folder.
 	-Open the terminal (make sure nodejs & angular is installed)
	-run ***npm i***
 	-run ***npm test***
